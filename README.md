## Main feature in project

- Board Detail Page
  - Drag and drop cards between columns
  - Add new column
  - Add new card
  - Light/Dark mode

## Technical Stack

- Framework: NextJS 14
- UI / CSS Library: MUI
- State Management: Tanstack Query v5
- Form Management: React Hook Form
- API: Rest APIs of repo trello-api
- More...
